# QUICKSTART — 5분 만에 시작하기

처음 쓰는 분을 위한 가장 짧은 안내입니다. 개발 지식이 없어도 따라 할 수 있습니다.

## 이게 뭔가요?

MBTI 16가지 성격 유형이 마치 회의실에 모인 것처럼, 당신이 던진 주제를 각자의
시각으로 평가하고 토론해 주는 **claude.ai용 도구**입니다. "여러 사람한테
물어본 효과"를 혼자서 얻을 수 있습니다.

- 한 명한테만 물어보기 → 한 유형의 의견
- 둘이 토론 붙이기 → 두 유형의 디베이트
- 회의 전체 → 16유형의 평가 + 정리된 결론

## 준비물

- **claude.ai 유료 플랜 계정** (Pro / Max / Team / Enterprise 중 하나)
- **컴퓨터 1대** — 업로드용 파일(zip)을 만들 때만 필요

## 3단계

### 1단계 — 스킬 파일(zip) 만들기

`claude-ai-skill` 폴더 안에서 zip 파일 하나를 만들면 됩니다.

- **터미널을 쓸 수 있다면**: `claude-ai-skill` 폴더로 이동해
  `bash build-zip.sh` 를 실행하세요. → `mbti-16types.zip` 이 만들어집니다.
- **터미널이 어렵다면**: 파일 탐색기에서 `claude-ai-skill/mbti-16types`
  폴더를 마우스 오른쪽 클릭 → **압축**(zip)을 선택하세요.
  (폴더 자체를 통째로 압축하는 것이 중요합니다.)

### 2단계 — claude.ai에 올리기

1. [claude.ai](https://claude.ai) 에 로그인
2. 왼쪽 아래 **설정(Settings)** 클릭
3. **Capabilities(기능)** → **Skills(스킬)** 메뉴로 이동
4. **Upload skill(스킬 업로드)** 버튼을 누르고 1단계에서 만든 zip을 선택
5. 목록에 `mbti-16types` 가 보이고 **켜짐** 상태이면 끝

### 3단계 — 써보기

새 대화창에서 **평소처럼 말로** 부탁하면 됩니다. 슬래시 명령 같은 걸 외울
필요가 없습니다.

처음이라면 아래 문장을 그대로 복사해서 보내 보세요:

```
16유형 전체 관점에서, "주말에 등산 갈까 집에서 쉴까"를 평가해줘
```

16가지 성격이 각자 점수와 의견을 내고, 마지막에 정리된 결론까지 나옵니다.

## 더 써먹는 법

| 하고 싶은 것 | 이렇게 말하면 됩니다 |
|---|---|
| 한 유형 의견만 | `ENFP라면 이걸 어떻게 볼까?` |
| 두 유형 토론 | `INTJ랑 ENFP를 토론시켜줘, 주제는 ○○` |
| 빠르게 / 깊게 | 문장에 `간단히` 또는 `깊게` 를 넣으면 분량이 조절됩니다 |
| 결과 저장 | `결과를 파일로 저장해줘` 라고 덧붙이면 됩니다 |
